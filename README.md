# Stakeholder-Brief
This project investigates the causes of declining performance across a 30-store retail chain over an 18-month period. The Analytics Team used structured AI prompting — built on the RCTTCE framework — to extract actionable insights from a ~50,000-record dataset covering store performance, product sales, customer transactions, and loyalty programme activity.
The goal: give the CEO clear, data-driven answers about what is going wrong and where the biggest opportunities for recovery lie.

in solving this assignment, i used AI to generate a prompt to identify problems and way out for a CEO who's retail company is declining in sales.
I Created my AI prompt using the RCTTCE framework. my prompt clearly include: 
R – Role: Define the specific role the AI should assume.
C – Context: Provide the business situation and available data.
T – Task: Specify exactly what the AI should produce. 
F – Format: Describe how the response should be structured.
Co – Constraints: State any limitations or requirements.
E – Examples: Include at least one example to guide the AI.



Using my RCTTCE prompt, i generated a one-page Stakeholder Brief for the CEO. and the  brief includes:

Situation Summary
Three Key Business Questions
Decisions These Questions Will Inform
Description of Available Data
